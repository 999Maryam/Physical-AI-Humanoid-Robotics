# Quickstart: UI Upgrade for Docusaurus Book and Chatbot

## Development Setup

1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run start
   ```

4. Visit `http://localhost:3000` to see the upgraded UI

## Key Files to Review

- `src/css/custom.css` - Global styling and theme variables
- `src/components/Chatbot.module.css` - Chatbot component styling
- `src/components/FloatingChatbot.module.css` - Floating chatbot styling

## Testing the Changes

1. Verify the new color scheme and typography across all pages
2. Test chatbot functionality and UI in both light and dark modes
3. Confirm responsive behavior on different screen sizes
4. Check all interactive elements have proper hover and focus states
